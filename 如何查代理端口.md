# 怎么查代理端口

代理端口在你用的「科学上网」软件里可以看到，常见软件位置如下。

---

## 飞梭 VPN（FlyShadow）

1. **打开飞梭客户端**，保持「已连接」状态。
2. 在软件里找 **「选项」「设置」或「参数」**（有的版本在任务栏右键图标里）。
3. 在设置窗口里找 **「本地端口」「代理端口」「HTTP 代理端口」** 等，记下数字（常见是 **1080**、**10809**、**7890** 等）。
4. 若没有「本地端口」：  
   - 看是否有 **「手动代理」** 或 **「HTTP/Socks5 代理」**，连接成功后一般会显示主机和端口。  
   - 或先试 **1080**、**10809**，在 **push-with-proxy.bat** 里改端口多试几次。

---

## Clash（小猫咪 / Clash for Windows）

1. 打开 Clash，点 **Settings / 设置** 或主界面 **General**。
2. 看 **Port** 或 **Mixed Port**：常见是 **7890**。
3. 有的版本在 **Settings → Mixin → 端口** 或主界面底部小字。

---

## V2rayN

1. 打开 V2rayN，顶部菜单 **参数设置**。
2. 看 **本地监听端口**：  
   - **Http 代理端口** 常见是 **10809** 或 1080。  
   - 填到 Git 时用这个 **Http** 端口。

---

## 其他常见软件

| 软件 | 常见端口 | 在哪看 |
|------|----------|--------|
| Clash | 7890 | 设置 / General 里的 Port、Mixed Port |
| V2rayN | 10809 或 1080 | 参数设置 → 本地监听端口（Http） |
| Shadowsocks | 1080 / 10808 | 系统代理或设置里的「本地端口」 |
| 其他 | 8080、7891 等 | 设置 / 偏好设置里的「端口」「Port」 |

---

## 用系统代理时

如果你在 Windows 里设置了「使用代理服务器」：

1. **Win + I** 打开设置 → **网络和 Internet** → **代理**。
2. 看「手动设置代理」里的 **地址**，一般是 `127.0.0.1`，后面会写 **端口**，例如 `7890`，那个就是端口号。

---

## 不确定就多试几个

很多人用的默认端口是：**7890**、**10809**、**1080**、**8080**。

可以先用 7890，在「选股网站」文件夹里双击 **push-with-proxy.bat**（脚本里默认是 7890）。  
若推送还是报错，用记事本打开 **push-with-proxy.bat**，把里面的 **7890** 改成 **10809** 或 **1080** 再保存，重新双击试一次。
