# A股选股网站

独立项目，与「电商客服与退换货管理系统」无关联。实时筛选 A 股，支持三种策略组合条件。

## 策略说明

- **策略一**：周 K 线缩量下跌，股价位于十日线以上，散户数量为负数（股东人数减少）。
- **策略二**：周 K 线量能变化在 90%～120%（轻微放量 / 平量 / 缩量）。
- **策略三**：股价位于五日线、十日线以上，大幅缩量下跌（量能缩小 50% 以上），散户数量为负数。

## 运行方式

```bash
cd 选股网站
npm install
npm run dev
```

浏览器打开 http://localhost:5174 ，在页面选择「扫描池数量」后点击「开始筛选」。数据通过东方财富公开接口获取，开发环境下已配置 Vite 代理以减轻跨域问题。

## 公网部署（手机可访问）

部署到 **Vercel** 后，可获得 HTTPS 链接，手机浏览器直接打开即可使用。

### 方式一：Vercel 网页一键部署（推荐）

1. 把本项目推到 **GitHub**（新建仓库后 `git push`）。
2. 打开 [vercel.com](https://vercel.com)，用 GitHub 登录。
3. 点击 **Add New → Project**，选择该仓库，**Framework Preset** 选 **Vite**，点 **Deploy**。
4. 部署完成后会得到 `https://xxx.vercel.app`，用手机浏览器访问即可。

项目里的 `vercel.json` 已配置好接口反向代理（东方财富、新浪），无需再改。

### 方式二：本机用 Vercel CLI 部署

```bash
npm i -g vercel
cd 选股网站
npm run build
vercel
```

按提示登录或绑定账号，然后按回车使用默认设置。部署成功后会输出线上地址。

### 说明

- 数据请求在部署后走同域 `/api/*`，由 Vercel 转发到东方财富/新浪，避免跨域。
- 手机访问时建议用浏览器「添加到主屏幕」，可当简易 App 使用。

## 技术栈

- React 18 + TypeScript
- Vite 5
- 数据来源：东方财富（列表、K 线、股东人数变化）

## 说明

- 股东「散户数量为负数」在本站中表示为**股东人数变化为负**（即股东人数减少）。
- 扫描数量越大耗时越长，建议先用 80～100 只做测试。
